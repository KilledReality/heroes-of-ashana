# Герои Асханы

Локальный прототип портала кампании для настольно-ролевой игры на базе Pathfinder 1e.

## Запуск

Откройте `index.html` в браузере. Сервер и установка зависимостей не нужны.

## Админ-доступ

Пароль прототипа: `ashana-master`

Данные сохраняются в `localStorage` браузера. Это удобно для первой версии, но не заменяет настоящую авторизацию и базу данных для публичного сайта.

## Что уже есть

- Wiki с публичными и мастерскими записями.
- Главная "мапа" Wiki с категориями: боги, места, NPC, государства, фракции, артефакты и другое.
- Отдельные страницы категорий Wiki с поиском внутри категории.
- Теги Wiki и фильтр по тегам.
- Загрузка картинок в Wiki-страницы через админ-режим, выбор формата, вписывания/обрезки, позиции и масштаба кадра.
- Интерактивная карта с масштабированием и маркерами.
- Гексовая карта-редактор: пустое полотно, раскраска гексов местностью, объекты на каждом гексе, публичные и GM-заметки.
- Админ может загрузить картинку в конкретный гекс; она автоматически подгоняется под форму гекса.
- Сайт автоматически подхватывает красивые тайлы местности из `assets/hex-terrain/` по заранее заданным именам файлов.
- Старые изображения Межей сохранены в `assets/maps/` как референсы, но основная карта теперь строится из редактируемых гексов.
- Расширенные листы персонажей Pathfinder 1e с полным списком навыков из `skills.txt`, языками, боем, магией, инвентарем и JSON-редактором.
- Галерея визуальных материалов.
- Доска заданий со скрытыми GM-записями.
- Roll-меню с формулами вида `1d20+5`, выбором персонажа, журналом бросков и бросками из листа персонажа.
- Админ-режим для редактирования, удаления, экспорта и импорта данных кампании.
