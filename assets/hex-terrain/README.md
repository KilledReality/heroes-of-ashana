# Hex Terrain Tiles

Сайт сейчас подхватывает PNG/WebP-картинки гексов местности с такими именами:

- `grass.png` — равнина
- `forest.png` — лес
- `water1.png` — вода
- `water bereg1.png` — вода: берег 1
- `water bereg2.png` — вода: берег 2
- `water bereg3.png` — вода: берег 3
- `water bereg4.png` — вода: берег 4
- `boloto.png` — болото
- `mountain.png` — горы
- `grass doroga1.png` — дорога
- `village.png` — город / деревня
- `small village.png` — малая деревня
- `castle.png` — укрепление
- `bad place.png` — опасность
- `temple destroyed.png` — подземелье

Дополнительные файлы, которые пока лежат как запасные варианты:

- `water2.png`

После обновления страницы гексы соответствующего типа будут автоматически использовать эти картинки.
